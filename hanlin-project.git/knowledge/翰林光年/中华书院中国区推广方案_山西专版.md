## 🎯 核心产品矩阵

### 1. 国际高中项目（主推）

**适用对象：**
- 初中毕业生（应届/往届均可）
- 无年龄、学历、成绩要求
- 适合华裔学生及移民家庭子女

**课程体系：**
```
├── 中文授课体系 (人教版 + 暨大版教材)
├── 英语强化课程
└── 升学规划服务
```

**优势承诺：**
- ✅ **100% 录取保障** - 双向承诺书制度
- ✅ **全额退费机制** - 毕业未被录取即退还学费
- ✅ **985/211 优先推荐** - 成绩优秀者冲刺清华北大等顶级名校

### 2. 华侨生升学规划服务

**服务内容：**
- 政策解读与申请指导
- 联考备考课程（每年 3 次考试机会）
- 志愿填报技术支持

**录取路径：**
```
第一阶段：初中毕业 → 中华书院高中入学
第二阶段：高中三年学习 + 华侨生联考备考
第三阶段：参加高考（华侨生类别）→ 985/211 录取
```

### 3. 大学申请直通车

**合作院校：**
- 暨南大学（国内直录）
- 海外名校（美国、英国、澳洲等，语言成绩可减免）

## 📋 入学流程（标准化 SOP）

### Step 1：咨询与评估
```
填写申请表 → 提交初中成绩单 → 免费学业评估 → 
初步方案制定
```

### Step 2：报名锁定名额
- 缴纳报名费 ¥3,000 RMB（可退）
- 签订《入学意向书》
- 获取招生顾问一对一服务

### Step 3：注册学籍与签证
- **菲律宾高中学历注册** - 双认证（菲教育部 + 中国驻菲大使馆）
- 中国留学签证 F 类办理支持
- 录取通知书发放

### Step 4：入学准备
- 行前培训课程（线上/线下可选）
- 住宿安排指导（校内宿舍或本地公寓推荐）
- 保险购买建议

## 🏢 合作模式选项

### Option A: 国际班共建（推荐）

**适用对象：** 现有民办中学/国际学校  
**服务内容：**
- 设立国际特色班/华侨生升学班
- 共享中华书院课程体系与师资
- 联合招生运营支持

**优势：**
- 快速启动项目（1-2 个月）
- 降低运营成本（共享场地/设施）
- 品牌背书增强家长信任度

### Option B: 独立招生代理

**适用对象：** 教育咨询机构/留学服务机构  
**服务内容：**
- 区域市场推广授权
- 招生咨询及客户管理
- 项目服务标准化支持

**优势：**
- 独家代理权（特定区域）
- 高额佣金激励（15%-20%）
- 品牌联合推广资源

### Option C: 城市合作伙伴

**适用对象：** 本地教育运营团队  
**服务内容：**
- 区域市场开发支持
- 招生渠道建设指导
- 项目落地运营协助
