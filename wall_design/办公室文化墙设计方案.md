# 中华书院·菲律宾帕赛分校  
## 办公室文化墙设计方案

---

**设计日期**: 2026-06-25  
**适用区域**: 前台接待区、办公走廊、员工休息区  
**设计理念**: 国际化视野 + 中华文化传承 + 教育专业性  

---

## 📐 总体布局规划

### 墙面分区（从左到右）
```
┌─────────────────────────────────────────────────────────────┐
│                                                                │
│   [Logo]    [中华书院介绍]   [留学生案例]   [教育理念]   [业务内容]  │
│    板块        板块              板块           板块         板块       │
│                                                                │
└─────────────────────────────────────────────────────────────┘
```

### 尺寸建议（单面墙）
- **总宽度**: 400cm × 高度：250cm
- **每个板块宽度**: 80cm（Logo:60cm, 介绍:100cm, 案例:80cm, 理念:80cm, 业务:70cm）
- **材质**: 亚克力 + LED 背光 + 铝合金边框

## 📋 制作清单

### 物料准备
- [ ] LED 背光灯箱 1 套（400cm×250cm）
- [ ] 亚克力面板 5 块（对应 5 个板块）
- [ ] 铝合金边框 1 套
- [ ] Logo 印刷品 3 份
- [ ] 学校建筑照片 3 张（高清，每张≥8000×6000px）
- [ ] 留学生案例照片/插画 5-8 张
- [ ] LED 电源及控制系统

### 安装步骤
1. **测量墙面**：确认尺寸和电源位置
2. **制作面板**：按设计图切割亚克力板
3. **打印内容**：UV 打印高清图文
4. **组装边框**：铝合金框架拼接
5. **安装调试**：LED 灯光调试

### 预算估算（参考）
- LED 灯箱：¥8,000-12,000
- 亚克力面板：¥3,000-5,000
- 印刷内容：¥2,000-4,000
- 边框及安装：¥3,000-6,000
- **总计**: ¥16,000-27,000
