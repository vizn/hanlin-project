# 区域墙面布局方案 (最终版 - 2026-06-26)

## 📏 尺寸规格
| 墙面 | 高度×宽度 |
|------|----------|
| **A（主背景墙）** | 240cm × 300cm |
| **B（辅助墙 - 案例区）** | 240cm × 600cm |
| **C（辅助墙 - 理念区）** | 240cm × 600cm |

## 🎯 墙面 A（主背景墙：Logo）
- **核心内容**: Logo + 品牌标语
- **设计定位**: 视觉焦点，强化品牌识别度
- **建议工艺**: 
  - Logo 采用亚克力发光字或立体烤漆工艺
  - 背景可搭配暗调色板提升质感
  - 内置 LED 灯带（色温 3200K）

### 内容布局：
- Logo 居中约占墙面积 40%
- 品牌标语占 20%："翰林光年"
- 副标题占 15%："一家专注于国际教育咨询的专业机构"
- 留白/装饰元素占 45%

## 🎯 墙面 B（辅助墙：留学生案例 + 学校介绍区）
- **核心内容**: 
  - 左侧（约 360cm 宽）：留学生成功案例展示
  - 右侧（约 240cm 宽）：**中华书院**等合作院校介绍/资质证明

### 中华书院（PPCHA）简介（墙面 B 右侧区）：
#### Philippine Pasay Chung Hua Academy（菲律宾帕赛中华书院）
- **机构定位**: 私立、非营利性、非宗派教育机构
- **办学宗旨**: 培养全面发展的个体，适应专业环境
- **教育理念**: 
  - PROVIDE（提供）优质教育
  - PROMOTE（促进）文化融合
  - CULTIVATE（培养）创新思维
  - HELP（帮助）学生成长
  - ACHIEVE（成就）未来目标

#### 教育阶段：
- 🎒 **学前教育** (Ching Chay Tho Memorial Preschool)
  - 开发孩子的好奇心、自信心、独立性与资源能力
  
- 📚 **小学教育** (Elementary Education)
  
- 🏫 **中学教育** (Secondary Education)

#### 特色项目：
- 🇨🇳 **中文语言艺术课程**（独特项目，体验中华文化）
- 💻 **计算机教育项目**（熟悉科技概念与应用）
- 🌍 **国际课程融合**

### 建议分区: 
- 案例区：每行展示 3-4 个案例卡片（含照片 + 简短说明）
- 学校区：中华书院专区 + 其他合作院校，按国家/地区分组

## 🎯 墙面 C（辅助墙：教育理念 + 业务内容）
### **上半部分**（约 120cm 高：教育理念）

#### 我们的核心理念
> "教育不是越早出国越好，而是越合适越好"
> 
> - **以学员为中心**，个性化定制方案
> - **降低试错成本**，先体验→再观察→再决定
> - **理性看待国际教育**，不盲目跟风

#### 三条核心理念：
1. ✅ **教育不是越早出国越好，而是越合适越好**
   - 国际教育不是"逃离"，而是更适合某些孩子的成长方式
   
2. ✅ **低龄阶段，更重要的是认知与适应，而非成绩**
   - 对 9-15 岁的孩子而言，真正重要的是：
     - 是否敢开口交流
     - 是否理解不同文化
     - 是否具备基本的独立与规则意识

3. ✅ **降低试错成本，是对家庭最大的负责**
   - 相比一次性、高成本、不可逆的选择
   - 我们更倾向于可控范围内的逐步探索

### **下半部分**（约 120cm 高：业务范围）

#### ✓ 一、学历教育项目
- 初中、高中、大学预科
- **中华书院**等合作项目

#### ✓ 二、体验项目
- 短期插班
- 国际课程体验营
- 校园真实嵌入式体验
- 语言环境浸泡与文化适应

#### ✓ 三、升学规划服务
- 国际升学就业规划全周期支持
- 中学衔接建议
- 本科及以上阶段规划
- 连贯、可持续的教育路径设计

## ⚠️ 注意事项
1. 墙面 A/B/C 需保持视觉连贯性，避免风格割裂
2. 灯光系统统一规划，建议色温≤3500K（暖白光）
3. 所有文字内容需预留多语言版本接口（中/英）
4. 案例区考虑后期更新机制（如磁吸式展板或电子屏）
5. **玻璃材质处理**：双层夹胶钢化，背发光亚克力板
6. 总墙面面积：A(72㎡) + B(144㎡) + C(144㎡) = **360㎡**
