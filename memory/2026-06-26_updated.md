# 区域墙面布局方案 (最终版 - 2026-06-26)

## 📏 尺寸规格
| 墙面 | 高度×宽度 |
|------|----------|
| **A（主背景墙）** | 240cm × 300cm |
| **B（辅助墙 - 案例区）** | 240cm × 600cm |
| **C（辅助墙 - 理念区）** | 240cm × 600cm |

## 🎯 墙面 A（主背景墙：Logo）
- **核心内容**: Logo + 品牌标语
- **设计定位**: 视觉焦点，强化品牌识别度
- **建议工艺**: 
  - Logo 采用亚克力发光字或立体烤漆工艺
  - 背景可搭配暗调色板提升质感
  - 内置 LED 灯带（色温 3200K）

### 内容布局：
- Logo 居中约占墙面积 40%
- 品牌标语占 20%："一家专注于国际教育咨询的专业机构"
- 留白/装饰元素占 40%

## 🎯 墙面 B（辅助墙：留学生案例 + 学校介绍区）
- **核心内容**: 
  - 左侧（约 360cm 宽）：留学生成功案例展示
  - 右侧（约 240cm 宽）：合作院校介绍/资质证明
- **设计定位**: 增强信任背书，展现实力与成果

### 建议分区: 
- 案例区：每行展示 3-4 个案例卡片（含照片 + 简短说明）
- 学校区：按国家/地区分组，配院校 Logo + 认证标识

## 🎯 墙面 C（辅助墙：教育理念 + 业务内容）
### **上半部分**（约 120cm 高：教育理念）

#### 我们的核心理念
> "教育不是越早出国越好，而是越合适越好"
> 
> - **以学员为中心**，个性化定制方案
> - **降低试错成本**，先体验→再观察→再决定
> - **理性看待国际教育**，不盲目跟风

#### 三条核心理念：
1. ✅ **教育不是越早出国越好，而是越合适越好**
   - 国际教育不是"逃离"，而是更适合某些孩子的成长方式
   
2. ✅ **低龄阶段，更重要的是认知与适应，而非成绩**
   - 对 9-15 岁的孩子而言，真正重要的是：
     - 是否敢开口交流
     - 是否理解不同文化
     - 是否具备基本的独立与规则意识

3. ✅ **降低试错成本，是对家庭最大的负责**
   - 相比一次性、高成本、不可逆的选择
   - 我们更倾向于可控范围内的逐步探索

### **下半部分**（约 120cm 高：业务范围）

#### ✓ 一、国际教育路径咨询
- 中小学国际教育方向评估
- 亚洲及其他经济型留学路径规划
- 国际课程与非主流路线对比分析

#### ✓ 二、国际教育体验与考察项目
- 校园真实嵌入式体验
- 语言环境浸泡与文化适应
- **短期插班**、**国际课程体验营**

#### ✓ 三、长期升学与转段支持
- 中学衔接建议
- 本科及以上阶段规划
- 连贯、可持续的教育路径设计
- **学历教育：初中、高中、大学预科**

#### ✓ 四、国际升学就业规划全周期服务
- 留学申请指导
- 签证办理协助
- 行前准备培训
- 海外生活适应支持

## ⚠️ 注意事项
1. 墙面 A/B/C 需保持视觉连贯性，避免风格割裂
2. 灯光系统统一规划，建议色温≤3500K（暖白光）
3. 所有文字内容需预留多语言版本接口（中/英）
4. 案例区考虑后期更新机制（如磁吸式展板或电子屏）
5. **玻璃材质处理**：双层夹胶钢化，背发光亚克力板
6. 总墙面面积：A(72㎡) + B(144㎡) + C(144㎡) = **360㎡**

- **品牌色卡**：科技蓝 #2196f3、活力橙 #ff9800、淡蓝灰 #e3f2fd
- **工艺建议**：亚克力发光字（厚度 15-20mm）+ LED 灯带
