# EducationStudio Workflow Guide

This file explains how to chain CreativeDirector, CopyWriter, ComfyUIDesigner, PPTDesigner, and BrandManager subagents.

## Subagent Spawning

### CreativeDirector (创意总监)
```
Tool: sessions_spawn
task: |
  ## Role
  You are CreativeDirector. Read skills/CreativeDirector/SKILL.md.

  ## Task
  [project brief]

  ## Knowledge
  Read: EducationStudio/knowledge/<project>/
  Read: EducationStudio/knowledge/品牌规范/

  ## Output
  Save creative brief to: outputs/Markdown/<project>_creative_brief.md
```

### CopyWriter (文案)
```
Tool: sessions_spawn
task: |
  ## Role
  You are CopyWriter. Read skills/CopyWriter/SKILL.md.

  ## Writing Brief
  [creative brief or file ref]

  ## Image Markers
  Mark image positions with: ![生成图片: English description of needed image]

  ## Output
  Save to: outputs/Markdown/<project>_<type>.md
```

### ComfyUIDesigner (AI绘图师)
```
Tool: sessions_spawn
task: |
  ## Role
  You are ComfyUIDesigner. Read skills/ComfyUIDesigner/SKILL.md.

  ## Task
  Read the copy file: outputs/Markdown/<project>_<type>.md
  Extract all ![生成图片: ...] markers
  Generate images using the appropriate workflow
  Save image map to: outputs/Markdown/<project>_image_map.json
```

### PPTDesigner (格式输出)
```
Tool: sessions_spawn
task: |
  ## Role
  You are PPTDesigner. Read skills/PPTDesigner/SKILL.md.

  ## Task
  Read: outputs/Markdown/<project>_<type>.md
  Run: python3 scripts/export_all.py <input.md> outputs/<Project>/<Project>
  
  ## Brand Template
  Read: scripts/brand_template.json

  ## Output
  - outputs/DOCX/<project>_<type>.docx
  - outputs/PDF/<project>_<type>.pdf
  - outputs/PPT/<project>_<type>.pptx
```

### BrandManager (品牌审核)
```
Tool: sessions_spawn
task: |
  ## Role
  You are BrandManager. Read skills/BrandManager/SKILL.md.

  ## Task
  Review the following deliverable for brand compliance:
  [file path]

  ## Use Checklist
  Run the full brand review checklist before approving.

  ## Output
  Save brand review report to: outputs/Markdown/<project>_brand_review.md
```

## Standard Project Types

### 宣传册 (Brochure) — Full Pipeline
1. **CreativeDirector** → creative brief
2. **CopyWriter** → brochure copy with `![生成图片: ...]` markers
3. **ComfyUIDesigner** → generate images → image_map.json
4. **BrochureDesigner** → assemble final markdown
5. **PPTDesigner** → export to DOCX + PDF + PPT via export_all.py
6. **BrandManager** → brand review → approve/reject

### 项目说明书 (Project Description)
1. **CreativeDirector** → outline sections
2. **CopyWriter** → write description
3. **PPTDesigner** → export to all formats
4. **BrandManager** → brand review

### 海报 (Poster)
1. **CreativeDirector** → visual direction
2. **ComfyUIDesigner** → generate hero background
3. **CopyWriter** → write headline + body matching image
4. **PPTDesigner** → export single-slide PDF

### PPT Deck
1. **CreativeDirector** → outline presentation structure
2. **CopyWriter** → write slide content
3. **PPTDesigner** → md2ppt.py → branded PPTX
4. **BrandManager** → brand review

## Image Workflow Mapping

| Image Need | Workflow JSON | Resolution |
|-----------|--------------|------------|
| Brochure cover/inside | Brochure/brochure_workflow.json | 512×768 |
| Poster background | Poster/poster_workflow.json | 512×768 |
| Campus scene | Campus/campus_workflow.json | 768×512 |
| Logo/icon | Logo/logo_workflow.json | 512×512 |
| PPT background | PPT/ppt_workflow.json | 1024×768 |

## Format Export

All formats are generated by `scripts/export_all.py`:
```bash
python3 scripts/export_all.py outputs/Markdown/<project>_<type>.md outputs/<Project>/<Project>
```

This creates:
- `outputs/<Project>/<Project>.docx` — branded DOCX
- `outputs/<Project>/<Project>.pdf` — branded PDF with header/footer
- `outputs/<Project>/<Project>.pptx` — branded PPT with accent bars

Individual scripts also available:
- `scripts/md2docx.py input.md output.docx`
- `scripts/md2pdf.py input.md output.pdf`
- `scripts/md2ppt.py input.md output.pptx`

## Brand Template
`scripts/brand_template.json` controls:
- Colors: 科技蓝 #2196F3, 活力橙 #FF9800, 淡蓝灰 #E3F2FD
- Fonts, sizes, margins for each output format
- PPT accent bars and slide dimensions

## Output Naming
- Copy: `outputs/Markdown/<project>_<type>_copy.md`
- Image map: `outputs/Markdown/<project>_image_map.json`
- Final (combined): `outputs/Markdown/<project>_<type>_final.md`
- Brand review: `outputs/Markdown/<project>_brand_review.md`
- DOCX/PDF/PPT: `outputs/<Format>/<project>_<type>.<ext>`
