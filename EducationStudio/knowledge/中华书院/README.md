# 中华书院（PPCHA）— 知识库

## 项目简介
**菲律宾帕赛中华书院**（Philippine Pasay Chung Hua Academy, PPCHA）—— 1932年成立，位于菲律宾马尼拉帕赛市的私立非宗派教育机构。
