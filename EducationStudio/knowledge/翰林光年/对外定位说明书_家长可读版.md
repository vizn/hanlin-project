## 🌟 什么是中华书院？

### 品牌背景

**中华书院 (China Book School)** 是菲律宾领先的国际教育机构，专注于为华裔学生提供：
- 优质基础教育
- 升学规划指导  
- 中华文化传承

**翰林光年**作为**中国区唯一授权合作伙伴**，致力于将这一优秀教育模式引入中国家庭。

### 办学理念

我们相信：
- **每个孩子都有自己的优势领域**

## 📝 入学流程说明

### Step-by-Step Guide

#### Phase 1: 咨询与评估（3-5 个工作日）

```
1. 填写在线申请表 (官网或小程序)
2. 提交初中成绩单/成绩证明
3. 招生顾问电话沟通 (了解孩子情况)
4. 免费学业评估报告 (发送至邮箱)
```

#### Phase 2: 报名与锁定名额（7 日内）

```
1. 签订《入学意向书》
2. 缴纳报名费 ¥3,000 RMB(可退)
3. 获取招生顾问一对一服务
4. 确认入学班级 (文科/理科班)
```

#### Phase 3: 注册学籍与签证（2-3 周）

```
1. 菲律宾高中学历注册完成
   - 双认证：菲教育部 + 中国驻菲大使馆
2. 录取通知书发放
3. 中国留学签证 F 类办理协助
4. 行前准备指导 (住宿/保险等)
```

#### Phase 4: 正式入学（每学期末）

```
1. 参加新生培训营 (线上/线下)
2. 领取教材与学习用品
3. 加入班级社群
4. 开启学习之旅
```

## 🎯 升学保障机制详解

### "100% 录取承诺"说明

**什么是双向承诺书？**

```
入学即签署《双向承诺书》:
├── 我们承诺：确保每位学生至少被一所大学录取
└── 您承诺：配合完成学业规划与备考任务

若毕业未被任何大学录取 → 退还全部学费
```

**"全额退费"范围：**
- 高中阶段所有已缴学费
- 教材费、注册费等
- **不包含：** 住宿/餐费（如自愿选择）

### 升学路径保障

> "即使不冲刺 985，也能进入 211 重点大学"

#### Level 2: 主推方案 - 985/211 冲刺

> "成绩优秀者冲刺清华、北大等顶级名校"  
> 录取率可达 85%+（优秀生源）

#### Level 3: 拓展方案 - 海外名校直通车

## ❓ 常见问题解答 (FAQ)

### Q1: 我的孩子成绩一般，能考上吗？
**A:** 我们有多元化升学路径。即使不冲刺顶尖名校，也能进入良好的大学。关键是找到适合孩子的路径。

### Q2: 需要英语基础好吗？
**A:** 不需要！我们的中文授课体系让语言学习更从容。同时提供雅思强化课程，提升国际竞争力。

### Q3: 如果孩子不适应国外生活怎么办？
**A:** 我们提供行前培训、住宿安排支持、心理辅导团队等全方位保障。首批学生已证明可成功适应。

### Q4: 费用是否可以分期付款？
**A:** 可以！学费支持按月/季度分期，减轻家庭一次性支付压力。具体方案请咨询顾问。

### Q5: 毕业后如果不想出国怎么办？
**A:** 完全没问题！985/211 学历同样受中国教育部认可，回国发展不受影响。许多学生选择回国就业或深造。
