# EducationStudio Image Prompt Templates

These templates provide consistent English prompts for common education imagery scenarios.
Use these when calling `comfyui_generate` for brochure and poster images.

## 1. Campus Exterior (校园外观)
```
modern school building exterior, clean architectural design, large glass windows,
green campus with trees and garden, bright sunny day, blue sky with white clouds,
professional architectural photography, high quality, sharp focus
```

## 2. Modern Classroom (现代化教室)
```
bright modern classroom interior, sunlight through large windows, wooden desks,
digital smart board on wall, plants in corner, clean organized space,
warm inviting atmosphere, professional interior photography
```

## 3. Students Studying (学生学习)
```
Asian high school students studying together at library table, focused expressions,
natural window light, books and laptops open, collaborative atmosphere,
soft warm tones, documentary photography style, candid moment
```

## 4. Teacher with Students (师生互动)
```
Asian teacher helping student with homework, one-on-one guidance in bright classroom,
warm supportive expression, educational setting, natural lighting,
professional educational photography, genuine interaction
```

## 5. Graduation Scene (毕业典礼)
```
graduation ceremony, students in caps and gowns throwing caps in air,
sunset golden hour lighting, joyful celebration, academic setting,
triumphant atmosphere, professional event photography
```

## 6. Abstract Education Concept (教育概念)
```
glowing lightbulb with book pages floating around, dark blue background,
knowledge tree growing from open book, magical educational concept,
soft blue and orange gradient lighting, clean minimalist 3D render
```

## 7. Student Dormitory (学生宿舍)
```
modern student dormitory interior, clean twin beds, study desks by window,
warm bedside lamp lighting, cozy comfortable atmosphere,
bright clean space, personal touches, professional interior photo
```

## 8. Campus Sports (校园运动)
```
students playing basketball on modern outdoor court, sunset golden light,
action shot, athletic youthful energy, green trees surrounding campus,
dynamic composition, professional sports photography
```

## Style Modifiers
Add to any prompt for consistent style:
- Brochure: `, professional brochure photography, soft studio lighting, warm inviting tone`
- Poster: `, dramatic lighting, high contrast, bold composition, attention-grabbing`
- Logo: `, clean flat vector design, solid background, minimalist, professional logo style`
- Campus: `, wide angle architectural photography, natural daylight, vibrant greenery`
