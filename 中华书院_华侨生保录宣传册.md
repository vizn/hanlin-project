## 📋 入学流程

### 📝 第一步：咨询与评估
```
填写申请表 → 提交初中成绩单 → 免费学业评估
```

### 💵 第二步：报名锁定名额
- 缴纳报名费 ₱28,500 PHP（约$500 USD）
- 签订《入学意向书》

### 🏫 第三步：注册学籍
- 注册菲律宾高中学历（双认证：菲教育部 + 留服中心认证）
- 领取录取通知书

### ✈️ 第四步：办理签证
- 中国留学签证 F 类
- 学校协助准备材料
